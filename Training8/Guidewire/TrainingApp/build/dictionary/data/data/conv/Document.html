Document (ab_document) (delegates to EventAware, Retireable)
Description
Internal representation of a physical or electronic document.
Entity: Document
Attributes: Editable?, Exportable?, Extendable?, Final?, Keyed?, Loadable?, Sourceable?, Versionable?
Messaging Events: The application will generate these events for this entity:
  • DocumentRemoved
  • DocumentAdded
  • DocumentStore
  • DocumentChanged
Integration programmers can write Event Fired rules to perform actions on these events. Refer to the Integration Guide for the complete list of events for all entities. Remember to configure a new or existing messaging destination in Studio to listen for desired events. Show user interface details on all fields
Fields

Author  varchar (60)     (exportable)?    (loadable)?    (writable)?
Name of the person who created the document.

DateCreated  datetime     (exportable)?    (loadable)?    (writable)?
Date and time the document was created.

DateModified  datetime     (exportable)?    (loadable)?    (writable)?
Date and time the document was last modified.

Description  shorttext (255)     (exportable)?    (loadable)?    (writable)?
Description of the document.

DMS  bit     (exportable)?    (loadable)?    (writable)?
Whether this document has content stored in a Document Management System.

DocUID  varchar (255)     (exportable)?    (loadable)?    (writable)?
The Unique Identifier (UID) for this document. The format of this UID is specific to the deployed Document Management System (DMS), and is passed to the configured IDocumentContentSource implementation.

DocumentIdentifier  varchar (60)     (exportable)?    (loadable)?    (writable)?
Short human-readable identifier for the document, often used as an extra storage location for form codes, when name and documenttype are inadequate.

Inbound  bit     (exportable)?    (loadable)?    (writable)?
Whether the document is an inbound, outbound, or stationary (null) document

Language  type key to LanguageType     (exportable)?    (loadable)?    (writable)?
The language in which this document is created.

MimeType  varchar (80)     (exportable)?    (loadable)?    (writable)?
The MIME type of this document; for example, application/msword for a Microsoft Word document.

Name  varchar (80)     (exportable)?    (loadable)?    (writable)?
Human-readable name of the document.

Obsolete  bit     (default: false)    (exportable)?    (loadable)?    (writable)?
If true, the information in the document can no longer be relied upon to be up-to-date and relevant. This is often used instead of deletion to preserve history.

PendingDocUID  varchar (255)     (exportable)?    (loadable)?    (writable)?
The document is pending, and it's pending storage has Unique Identifier (UID). The format of this UID is specific to the IDCS implementation.

PublicID  publicid (20)     (exportable)?    (loadable)?    (non-null)?    (writable)?
ID or primary key of the row in the external system to which this row is mapped

Recipient  varchar (60)     (exportable)?    (loadable)?    (writable)?
Name of the intended recipient of the document (if any).

Section  type key to DocumentSection     (exportable)?    (loadable)?    (writable)?
The section to which this document belongs, if any.

SecurityType  type key to DocumentSecurityType     (exportable)?    (loadable)?    (writable)?
Type of document used for access-restriction purposes, in conjunction with the information in security-config.xml.

Status  type key to DocumentStatusType     (exportable)?    (loadable)?    (writable)?
The current status of the document, if any.

Type  type key to DocumentType     (exportable)?    (loadable)?    (writable)?
The specific type of the document, if any.
