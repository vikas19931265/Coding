AddressBookConvertable (Virtual)
Description
Indicates whether a Contact entity is linkable between a Guidewire application and ContactManager. This is an internal system entity.
Entity: AddressBookConvertable
Attributes: Final?
Fields

AddressBookUID  varchar (30)     (writable)?
Represents the ID of the associated object in Address Book. Null if the object is not linked to Address Book.

LinkedContact  Derived property returning Contact     (virtual property)?
Deprecated
Gets the Address Book Contact to which this Contact is linked. Also brings related contacts in primary relationships with the linked contact.

Performance note: This operation requires a remote access to the Address Book.
