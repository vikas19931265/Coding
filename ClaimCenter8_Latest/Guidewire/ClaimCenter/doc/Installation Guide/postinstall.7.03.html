Setting Up Free-text Search for Embedded Operation

Set up free-text search for embedded operation for preliminary testing with small amounts of test data. For final testing with larger volumes of data and to prepare for production, set up free-text search for external operation.
To set up free-text search for embedded operation
1.
In Studio, open Configuration → config → solr → solrserver‑config.xml.
2.
Verify that the file contains a <solrserver> element that matches the following:
 
    <solrserver name="embedded" type="embedded">
      <param name="provision" value="true"/>
      <param name="solrroot” value="/opt/gwsolr"
    </solrserver>
3.
In the <document> element, change the servername attribute to embedded.
    <document name="claimcontact" archive="false" servername="embedded"/>
4.
Save your changes to the solrserver-config.xml file.
5.
Open a command prompt to ClaimCenter/bin and run the following command:
    gwcc solr
