ClaimCenter System Administration Guide : ClaimCenter Administrative Commands : Template Tools Command

Template Tools Command

template_tools -help
template_tools -password password [-server url] [-user user] [ -D name=value ]
               [ -working_dir name] { -convert_dir directory | -convert_file filename |
               -list_template | -import_dir objectsfile fieldsfile directory | 
               -import_files objectsfile fieldsfile outfile | -validate_all | -validate_template id }

Template Tools Options

You can use any of the following options with the template_tools command. You must always supply the -password option.
 
Option
Description
-help
Prints a help message.
-convert_dir directory
Converts templates in the specified directory to the new format.
-convert_file filename
Converts the specified template to the new format.
-import_dir objectsfile fieldsfile directory
Imports context objects and form fields into all the templates in the specified directory.
-import_files objectsfile fieldsfile outfile
Imports context objects and form fields into all the templates in the specified template outfile.
-list_templates
Lists all of the templates available for validation.
-password password
Specifies the administrative password. You must specify a password.
-server url
Specifies the ClaimCenter host server url.
-user user
Specifies the user to run the process.
-validate_all
Validates all the templates.
-validate_template id
Validates a single template.
-working_dir directory
Specify a directory so that relative paths can be used for file arguments.
