FNOL Mapper Command

fnol_mapper -help
fnol_mapper -password password [-server url] [-user user] {-input filename1  -mapping filename
            -property directory | -D name=value }
The fnol_mapper command accepts first notice of loss (FNOL) claim data in an XML file. A mapping class file contains a series of directives to transform the incoming FNOL data into a ClaimCenter Claim object. Then, ClaimCenter imports the object. The fnol_mapper command calls the IClaimAPI.importClaimFromXML method. See the Gosu documentation for details. Generate Gosu documentation by navigating to ClaimCenter/bin and running the gwcc regen-gosudoc command.

FNOL Mapper Options

You can use any of the following options with the fnol_mapper command. You must always supply the -password option.
 
-D name=value
Sets a Java system property.
-help
Prints a help message.
-input filename
Name of XML input file.
-mapping filename
Name of XML mapping file.
-password password
Specifies the password to use to connect to the server. ClaimCenter requires this option.
-property directory
Specifies directory containing all the property files, such as logging.properties and typelist property files.
-server url
Specifies the ClaimCenter host server url.
-user user
Specifies the user to run the process.
