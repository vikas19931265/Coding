Updating Customized Check Wizard

Guidewire has removed the custom check recurrence widget PaymentRecurrenceValueWidget and its supporting classes from ClaimCenter 7.0. If you have made changes to the Instructions step of the Check Wizard (config/web/pcf/claim/newtransaction/check/NewPaymentInstructionsDV.pcf), manually update your custom version of NewPaymentInstructionsDV.pcf to use the new CheckRecurrenceInputSet instead of PaymentRecurrenceInput.
Additionally, examine any custom Check Wizard PCF files you have created that reference obsolete check wizard widgets, and remove those references. For example, any references to NormalCreateCheckWizard produce an error at server startup. Studio does not flag these invalid widget references for you.
The full list of deleted elements is:
•
CreateCheckWizard
•
EditCheckWizard
•
EditRecurrenceCheckWizard
•
NormalCreateCheckWizard
•
ManualCreateCheckWizard.
•
QuickCreateCheckWizard
