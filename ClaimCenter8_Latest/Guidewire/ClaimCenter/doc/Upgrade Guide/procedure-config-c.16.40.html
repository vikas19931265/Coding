Running PCF Iterator Upgrade

Prior to ClaimCenter 7.0, toolbar buttons, with the exception of CheckedValues, did not specify an iterator. ClaimCenter used the first iterator defined in the PCF file following the button. In ClaimCenter 7.0, all iterator buttons have an iterator attribute set. Each toolbar button must have a specified iterator on which it operates.
ClaimCenter 7.0 includes a command-line tool to update PCF files to specify an iterator for toolbar buttons. The tool selects the first iterator defined in the PCF file following the button.
The tool only works on PCF files in the configuration module. You could have a PCF file that references a list view defined in another PCF file. However, the PCF file that defines the list view is unchanged from the default configuration and is therefore not in the configuration module. In this case, copy the PCF file containing the list view definition to the configuration module, preserving the directory structure.
To run the PCF iterator upgrade
1.
In the target ClaimCenter 8.0.1 directory, navigate to the bin directory.
2.
Run the following command:
gwcc iterator-upgrade
