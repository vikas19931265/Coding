Merging Files Related to ContactManager Integration

The following files might appear in the configuration upgrade tool for manual update after you complete the automatic upgrade. If your 6.0.x version of ClaimCenter is not integrated with ContactCenter (now called ContactManager), you can simply accept all these file changes:
•
ab-to-cc-data-mapping.xml
•
cc-to-ab-data-mapping.xml
•
search-filter.xml
•
ContactSearchCriteria.etx
•
contact-sync-config.xml
•
contact-relationship-config.xml
If your 6.0.x version of ClaimCenter is integrated with ContactCenter, be aware that there have been changes in the files used to integrate these two applications. Before updating any of these files, it is best to complete the ContactManager upgrade, as described at “Upgrading ContactManager from 6.0.x”.
After completing that upgrade, you can perform the upgrade described in “Upgrading ClaimCenter from 6.0.x for ContactManager”.
