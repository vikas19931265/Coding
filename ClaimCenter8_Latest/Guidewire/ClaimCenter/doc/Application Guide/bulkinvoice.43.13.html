Bulk Invoice Events and Acknowledgements

ClaimCenter generates a specialized BulkInvoiceStatusChanged event every time it updates the bulk invoice status, as well as when it first creates the bulk invoice. In addition to the BulkInvoiceStatusChanged event, ClaimCenter generates the following events as well:
•
BulkInvoiceAdded
•
BulkInvoiceUpdated
•
BulkInvoiceRemoved
These events are similar to the events that ClaimCenter generates for all financial transactions and checks.
IMPORTANT You must implement a version of the message transport plugin that can listen for the BulkInvoiceStatusChanged event. This message transport plugin implementation is necessary for ClaimCenter to be able to pass the check to an external system. See “Bulk Invoice Integration” in the Integration Guide for more information.
