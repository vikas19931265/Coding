Entity Methods loadByKey and loadByPublicID Deprecated, New Method loadBean

The Bundle class methods loadbyKey and loadByPublicID are deprecated in version 8.0.
Instead of using loadbyKey, use the new method loadBean, which takes a gw.pl.persistence.core.Key object. Create a Key object with a constructor that takes the entity type and the numeric ID. For example:
a = gw.transaction.Transaction.getCurrent().loadBean(new Key(Address, 123))
Instead of using loadbyPublicID, use the query builder APIs. For example:
Query.make(Claim).compare(Claim#PublicID, Relop.Equals, myPublicId).select().AtMostOneRow
