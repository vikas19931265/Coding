New Reinsurance Manager User Role

The new Reinsurance Manager user role has been constructed to be a standalone user role, enabling you to create a user and assign it just this one role. This Reinsurance Manager user is able to log into ClaimCenter and perform all tasks associated with reinsurance management, including:
•
Viewing and editing transactions on the new Reinsurance Financials Summary page.
•
Editing Reinsurance Recoverable setting and reason fields on the Claim Status page.
•
Viewing and editing Reinsurance Thresholds.
These tasks are embodied in the full set of Reinsurance permissions. One or all of these permissions can be added to existing roles. See “Changed Reinsurance Permissions”.
