ServiceRequest (cc_servicerequest) (delegates to Assignable, CCAssignable, Editable, EventAware, Extractable, Validatable)
Description
Represents a unit of work requested of a specialist or vendor, including instructions that describe the work to be done and the current state of the ServiceRequest.
Entity: ServiceRequest
Attributes: Editable?, Exportable?, Extendable?, Final?, Keyed?, Sourceable?, Versionable?
