ClaimValidationWorkItem (cc_validationworkitem) (delegates to KeyableBean, WorkItem)
Description
Represents a unit of validation work pending for a claim. Claims are bulk-validated using the distributed work queue. A ClaimValidationWorkItem entity is created for each Claim to be validated and the queue worker threads process work items as they become available.
