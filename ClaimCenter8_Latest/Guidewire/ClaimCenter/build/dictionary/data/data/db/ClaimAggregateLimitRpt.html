ClaimAggregateLimitRpt (cc_claimagglimitrpt) (delegates to AggregateLimitRptDelegate, Editable)
Description
Represents denormalized aggregate limit used calculations (i.e., the amount of the limit that has been used to date) for a claim as related to a given aggregate limit and optionally, for a coverageline.
