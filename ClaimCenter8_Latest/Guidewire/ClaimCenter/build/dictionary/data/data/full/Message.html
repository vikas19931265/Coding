Message (cc_message) (delegates to KeyableBean)
Description
getDestinationID. Messages carry a payload containing the data to be sent to the destination. Messages are dispatched using a MessageTransport plugin specified in config.xml, which is responsible for sending the message to the receiving system. See the ClaimCenter Integration documentation for more details about the lifecycle of Messages.
