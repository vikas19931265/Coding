Changes to Geocoding Service

ClaimCenter provides a new, fully implemented and supported implementation of the GeocodePlugin that connects with the Microsoft Bing Maps Geocode Service. In the default configuration of ClaimCenter, the GeocodePlugin is disabled but configured to use the BingMapsPlugin implementation. You must obtain a Bing Maps application key from Microsoft as credentials for this implementation.
For existing customers, Guidewire continues to support the MappointGeocodePlugin implementation of the GeocodePlugin that connects with the Microsoft MapPoint web service. However, Microsoft has announced the retirement of the MapPoint web service, on November 18, 2011. In its place, Microsoft offers its Bing Maps Geocode Service and recommends that MapPoint customers migrate to Bing Maps as soon as possible.
 
 Priority:
Before deploying a production server
 Summary:
Upgrade your Microsoft MapPoint account to a Bing Maps account and edit the GeocodePlugin registry to use the BingMapsPlugin implementation.
