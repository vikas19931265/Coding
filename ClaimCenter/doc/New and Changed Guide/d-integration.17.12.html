Escalation and Exception Plugins

There are several optional exception and escalation plugins. By default, they just call the associated rule sets and perform no other function. Implement your own version of the plugin and register it in Studio if you want something other than the default behavior. The following table lists the exception and escalation plugins:
 
Name
Plugin interface
The default action
Activity escalation
IActivityEscalationPlugin
Calls the activity escalation rule set
Group exceptions
IGroupExceptionPlugin
Calls the group exception rule set
User exceptions
IUserExceptionPlugin
Calls the user exception rule set
For more details, see “Exception and Escalation Plugins” in the Integration Guide.
