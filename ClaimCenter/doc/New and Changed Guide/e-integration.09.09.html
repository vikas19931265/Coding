Regenerating WSDL and XSD in Web Service Collections (regen-from-wsc)

Web service collection (.wsc) files encapsulate the set of resources necessary to connect to a web service on an external system. If you view a web service collection in Studio and click the Fetch Updates button, Studio gets the latest WSDL and XSD files from the servers that publish those web services. In this release, you can trigger the Fetch Updates process from a command line tool called regen-from-wsc. See “Loading WS-I WSDL Locally Using Studio Web Service Collections” in the Integration Guide.
