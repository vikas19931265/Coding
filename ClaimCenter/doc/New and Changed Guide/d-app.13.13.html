New Reinsurance Financial Summary Page

The Reinsurance link appears in the left-hand Claims navigation frame. It appears for all users with:
•
The Reinsurance role
•
All other users that have had the View RI Transactions and Agreements permission added to their role
This link opens the Reinsurance Financial Summary page.
The Reinsurance Financial Summary page is designed to show all reinsurance treaties and reinsurance recoverable amounts as they apply to individual claims. Within a claim, the reinsurance recoverable amounts are further broken down to the amount recovered and from which treaty or facultative agreement on a per exposure basis.
For more information about reinsurance concepts see “Reinsurance Management Concepts” in the Application Guide.
The Reinsurance Financial Summary page is available for all claims, whether reinsurance applies to a particular claim or not. The information on the Reinsurance Financial Summary page is retrieved from the reinsurance system. To retrieve this information, ClaimCenter must be integrated with the reinsurance system. This integration procedure is described in the Integration Guide.
The Reinsurance Financial Summary page will show there is no reinsurance for a claim in three cases:
•
There is no reinsurance treaty or facultative agreement that is available for that claim.
•
There are applicable reinsurance treaties and/or facultative agreements, but there are no exposures on the claim.
•
ClaimCenter has not been integrated with a reinsurance system.
The Reinsurance Financial Summary page will show which treaties and facultative agreements are applicable, or attached, to a claim only after an exposure on that claim has been created.
After a reserve line has been created on the exposure, reinsurance reserve line calculations automatically appear for the proportional treaties that attach to the claim.
After payments are made, the reinsurance recoverable per proportional reinsurance treaty automatically appears.
All non-proportional treaty and facultative reinsurance reserve line and recoverable amounts must be added manually. The user must readjust the automatic proportional calculations manually if they fall within the same range as manually applied non-proportional treaties.
Treaties and facultative agreements that appear on this page can be edited or added.
To change information on this page, the user must have the Reinsurance Manager Role or they must have had the Edit RI transactions and agreements permission added to their role.
