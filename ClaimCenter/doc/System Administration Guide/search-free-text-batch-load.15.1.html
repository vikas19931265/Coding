ClaimCenter System Administration Guide : Free-text Batch Load Command

Free-text Batch Load Command

The free-text batch load command loads the Guidewire Solr Extension, a full-text search engine, with index documents for all claim contacts in your ClaimCenter application database. Index documents are XML documents that contain a subset of the information from claim contacts in ClaimCenter. The Guidewire Solr Extension indexes the documents after it receives them from the free-text batch load command.
Note: The free-text batch load command runs on the host where the Guidewire Solr Extension resides. The command is located in the /opt/gwsolr/cc/solr/claimcontact_active/conf directory, not the ClaimCenter/admin/bin directory.
