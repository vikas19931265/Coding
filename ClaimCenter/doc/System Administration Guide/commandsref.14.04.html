ClaimCenter System Administration Guide : ClaimCenter Administrative Commands : Import Tools Command

Import Tools Command

import_tools -help
import_tools -password password [-server url] [-user user] [-ignore_all_errors]
             { { -import filename1, filename2 ... [{-dataset dataset | -output_csv filename | 
             -output_xml filename}] | -D name=value |
             -privileges } [-ignore_all_errors] [-ignore_null_violations] }
The import_tools command imports new or updated data into existing tables in the ClaimCenter database. You can only import data for valid entities or their subtypes. ClaimCenter supports this command for importing administrative data but not for importing other data into ClaimCenter. Instead, use staging tables or APIs to import other types of data into ClaimCenter.
Note: ClaimCenter does not fire any events related to the data you add or modify through this command.

Import Tools Options

You can use any of the following options with the import_tools command. You must always supply the -password option.
 
Option
Description
-D name=value
Sets a system property.
-dataset integer
Specifies the integer representing the dataset to import. Datasets are ordered by inclusion. The smallest dataset is always numbered 0. Thus, dataset 0 is a subset of dataset 1, and dataset 1 is a subset of dataset 2, and so forth. To import all data, set this value to -1.
-help
Displays the command usage.
-ignore_all_errors
Causes the command to ignore any errors in a CSV file.
-ignore_null_violations
Causes the command to ignore violations of null constraints in import data.
-import filename1, filename2, ...
Imports administrative data from a comma-separated list of CSV or XML files.
-output_csv filename
If used with the -import option, outputs CSV to the specified file and then stops processing. ClaimCenter imports no data into the server. Use this command to convert XML input files to CSV.
-output_xml filename
If used with the -import option, outputs XML to specified file and then stops processing. ClaimCenter imports no data into the server. Use this command to convert CSV input files to XML.
-password password
Specifies the password to use to connect to the server. ClaimCenter requires the password.
-privileges
Rebuilds role privileges by deleting role privileges in the current database and importing the roleprivileges.csv file in ClaimCenter/modules/configuration/config/import/gen.
-server url
Specifies the ClaimCenter host server url.
-user user
Specifies the user to run the process.
