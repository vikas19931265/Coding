Understanding the Language Module Installer

The language module installer is a command line tool that lets you install or upgrade installed display languages. The syntax for the command is:
gwcc install-localized-module -Dmodule.file=fileName.zip -Dinstall.type=installType
The command requires you to specify the following parameters.
 
Command Parameter   
Description
Dmodule.file
The name of the ZIP file that contains the language module. Be certain to include the .zip extension. The file must be in the root of the ClaimCenter home directory.
Dinstall.type
The type of operation for the language module installer to perform:
•
install – Install a language module for a language that is not currently installed
•
upgrade – Upgrade an installed language with a newer version of the language module
