Financial Transactions Outside the User Interface

If you create financial transactions in the user interface, ClaimCenter does all the bookkeeping for you. It adjusts aggregate limits, updates all T-accounts, changes all summary financial amounts, and so on. You can also use Gosu to create financial transactions, or you can use the Transaction Presetup rule set.
For more information, see:

“Transaction Set Presetup Rules” in the Rules Guide
•
“Creating Reserve Transactions Directly” in the Configuration Guide
