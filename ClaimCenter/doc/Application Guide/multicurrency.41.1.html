ClaimCenter Application Guide : ClaimCenter Financials : Multiple Currencies

Multiple Currencies

ClaimCenter can be set up to use a single, base currency or different currencies for all its financial transactions, based on your business needs. Enabling ClaimCenter to use multiple currencies, known as multicurrency, means that you can create reserves and recovery reserves, write checks, and make payments in more than one currency.
This topic describes how multicurrency works, the role of exchange rates, and the various uses of multicurrency in ClaimCenter.
