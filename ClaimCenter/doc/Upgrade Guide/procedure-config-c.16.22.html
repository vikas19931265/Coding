Capturing Location-based Risks Other than Buildings

ClaimCenter 7.0 added a LocationMiscRU entity that has an OtherRiskType field. You can use the OtherRiskType field to distinguish risk units that are not buildings or vehicles.

Updating PolicyLocation

In ClaimCenter 7.0, Guidewire edited the definition for PolicyLocation to add an array that stores a list of location-based risk units for the property. The PolicyLocation entity is defined at /config/metadata/PolicyLocation.eti.

Updated Risk Unit User Interface

In ClaimCenter 7.0, Guidewire created a new user interface for location-based risk units. Prior to ClaimCenter 7.0, the user interface contained one list view for risk units with optional buildings. As of ClaimCenter 7.0, the interface has two list views, one for each of the following:
•
risk units with buildings
•
other location-based risk units
