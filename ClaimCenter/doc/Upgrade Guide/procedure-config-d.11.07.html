Uninstalling Language Modules

If you have language modules installed, uninstall the language modules before you upgrade. After you have upgraded the configuration, you can reinstall language modules.
To uninstall the 7.0 language module
1.
Open modules/configuration/module.xml.
2.
Change the <dependency> from the language module to cc-gunit.
3.
Save modules/configuration/module.xml.
WARNING Your ClaimCenter 7.0 server will not start after uninstalling the language module.
