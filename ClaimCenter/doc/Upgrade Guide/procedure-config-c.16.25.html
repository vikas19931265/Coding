ClaimCenter Upgrade Guide : Upgrading from 6.0.x : Upgrading the ClaimCenter 6.0.x Configuration : Changes to ClaimInfoSearchView Entity

Changes to ClaimInfoSearchView Entity

Guidewire has removed the following fields from the ClaimInfoSearchView view entity:
•
ActiveAdjuster
•
ActiveInsured
•
ClaimCurrency
•
ClaimNumberOrder
•
ClaimStateOrder
•
Flagged
•
FuturePayments
•
LossDateOrder
•
PolicyNumberOrder
•
RemainingReserves
•
TotalPayments
In ClaimCenter 6.0.6 and 7.0.1, Guidewire made the following changes:
•
Removed references to the previously listed fields from ArchiveClaimRetrievePopup.pcf and ClaimSearchResultsLV.pcf.
•
Updated ArchiveClaimRetrievePopup.pcf to remove cells that refer to the RemainingReserve, FuturePayments, and TotalPayments fields.
•
Removed the Flagged column from the archived mode of ClaimSearchResultsLV.pcf.
•
Removed the display keys ArchivedClaim.FuturePayments.NoValue, ArchivedClaim.RemainingReserves.NoValue, and ArchivedClaim.TotalPayments.NoValue. These financial fields are no longer shown in archived mode of ClaimSearchResultsLV.pcf.
