Introduces the application, explains application concepts,
and provides a high-level view of major features and business
goals of the application. This is your first place to look
when learning about a feature, with links to other books as
appropriate for details. Everyone should read this book as an
overview of the application.
Provides instructions for upgrading ClaimCenter
from previous versions.
Describes new features and changes to features that existed
in prior ClaimCenter versions. This guide is a companion
to the Upgrade Guide.
Describes how to install a new copy of ClaimCenter
into Windows or UNIX environments. System administrators
and developers who need to install the ClaimCenter server
or any associated administrative tools should read this.
Provides guidance for the ongoing management of a ClaimCenter
system. This document is intended to help system administrators
monitor ClaimCenter, manage its security, and take care of
routine tasks such as system backups, logging, and importing files.
Documents how to configure ClaimCenter, including basic
steps and examples for implementing such configurations.
This is intended as a reference, not to be read cover-to-cover.
IT staff and system integrators should read this to configure
ClaimCenter for an initial implementation or for custom enhancements.
Describes how to configure ClaimCenter for a global environment.
Learn about global locales, languages, date and number formats,
names, currencies, addresses, and phone numbers. The intended
readers are configuration engineers who work with locales
and languages.
Describes the business rule methodology, rule categories for
ClaimCenter, and rule syntax for Guidewire Studio. All Gosu
business rule programmers and all analysts who will define the
business rule logic should read this.
A language reference guide for the syntax of expressions
and statements within ClaimCenter. This document also
provides examples of how the syntax can be constructed to
create rules within the system. Rule writers who create
and maintain rules in Guidewire Studio should read this.
Detailed documentation of the ClaimCenter data model,
including customer data model extensions. With a local
copy of the product, regenerate the dictionary by going
to the ClaimCenter/bin directory and running
the gwcc regen-dictionary command.
Documents all security permissions, roles, and the
relationships among them. With a local copy of the
product, regenerate the dictionary by going to the
ClaimCenter/bin directory and running the
gwcc regen-dictionary command.
A self-serve web site with a wide-range of resources
for Guidewire customers, partners, and field staff. It
includes a web-based issue entry and tracking system, a
knowledgebase with practical tips from experienced field
staff, and forums for networking and sharing best practices.
Provides integration programmers with an architecture
overview and "how to" information for integrating ClaimCenter
with external systems and custom code. The Integration Guide
is a critical learning tool for explanations and examples, but
directs programmers to relevant parts of the Java API Reference
Javadoc and SOAP API Javadoc (listed below) for further details.
All integration programmers and consultants should read this.
This document provides a number of best practices for installing, configuring, and operating an instance of ClaimCenter.
Documents the Java API for integration programmers:
  • Java plugin interface definitions. These also are
    the specs for Gosu plugins
  • ClaimCenter Java entity fields including both
    entity data and additional domain methods
  • Other Java utility classes.
With a local copy of the product, to regenerate these files:
Go to the ClaimCenter/bin directory and
run the gwcc regen-java-api command.
For RPCE web service, documents web services integration programmers.
With a local copy of the product, regenerate these files:
Go to the ClaimCenter/bin directory and
run the gwcc regen-soap-api command.
For WS-I web services, the WSDL is generated but not Javadoc-style HTML.
For WS-I web service details, see the implementation classes in Studio
or import the WSDL into your desired SOAP client language or SOAP client IDE.
Documents all types visible from the Gosu type system.
With a local copy of the product, regenerate these files:
Go to the ClaimCenter/bin directory and
run the gwcc regen-gosudoc command.
Explains how to manage contacts within ClaimCenter, and also
how to manage contacts centrally with ContactManager. System
integrators and administrators will find this book useful.
Definitions of terminology used in Guidewire documentation.
Describes the Guidewire Insurance Suite at a feature level
and an integration level.
