RICoding (cc_ricoding) (delegates to Extractable, Retireable, TAccountOwnerDelegate)
Description
Associates a Claim's Reinsurance Financial information with the appropriate RIAgreement and ReserveLine. Every Reinsurance Transaction must have an RICoding, thereby relating the Transaction with a specific RIAgreement (and hence Reinsurer) as well as an obligation on the Claim.
