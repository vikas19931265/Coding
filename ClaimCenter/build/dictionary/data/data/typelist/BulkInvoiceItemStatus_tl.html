BulkInvoiceItemStatus     (cctl_bulkinvoiceitemstatus)
Possible statuses for BulkInvoiceItem entities
Attributes: Final

Referenced by:
BulkInvoiceItem.Status
Typecodes
Code Name Description Priority Internal Retired
approved Item approved Bulk invoice item has passed the bulk invoice approval process   true  
awaitingsubmission Awaiting submission Bulk invoice item and its check have passed approval and is ready to be escalated once the bulk invoice is ready to be escalated   true  
checkpendingapproval Check pending approval Bulk invoice item has passed the bulk invoice approval process and is waiting for its check to be approved   true  
draft Draft Bulk invoice item is possibly committed to the database, but is not yet ready for validation   true  
inreview In review Bulk invoice item requires action before it can be paid. This value can be set when approving a bulk invoice.   true  
notvalid Not valid Payment/check associated with the bulk invoice item failed validation   true  
pendingstop Pending stop Bulk invoice item was stopped, and confirmation of stop is pending   true  
pendingtransfer Pending transfer Check associated with bulk invoice item was transferred, and acknowledgement of the transfer event message is pending   true  
pendingvoid Pending void Bulk invoice item was voided, and confirmation of void is pending   true  
rejected Rejected Bulk invoice item was rejected by the bulk invoice approver   true  
stopped Stopped Bulk invoice item was stopped   true  
submitted Submitted Bulk invoice item was submitted to the downstream system   true  
submitting Submitting Bulk invoice item is being submitted to downsteam system   true  
transferred Transferred Check associated with bulk invoice item was transferred   true  
voided Voided Bulk invoice item was voided   true
