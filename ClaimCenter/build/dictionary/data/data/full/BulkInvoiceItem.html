BulkInvoiceItem (cc_bulkinvoiceitem) (delegates to Retireable)
Description
Describes one item of the BulkInvoice, and corresponds to one line item of the original invoice. It contains data fields that define the reserve line to which the item is to be coded, as well as the amount and the PaymentType.
